# pokegraf

Pokegraf is a telegram bot made with ❤️ by [Lucas Maximiliano Marino](http://lucasmarino.me/).  

It uses the [PokeAPI](https://github.com/PokeAPI/pokeapi) and [Telegraf](https://github.com/telegraf/telegraf/).  

# Roadmap

- Inline behaviour
- Rework on routing
- Stats, abilities, etc.
